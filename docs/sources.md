# Sources

Initial public sources to investigate:

| Source | URL | Notes |
| --- | --- | --- |
| OpenPets | https://openpets.app/ | Registry with verified downloadable packs. |
| Codex Pets | https://codexpets.org/ | Gallery and install documentation. |
| CodingPets.dev | https://www.codingpets.dev/ | Generator and install metadata. |
| CodexPetHub | https://codexpethub.com/ | Hub that documents hash-verified installs. |
| codex-pet.com | https://codex-pet.com/ | Gallery with CLI add examples. |
| Pets Codex | https://petscodex.com/ | Community submission-oriented site. |

Investigation checklist:

- Does the source provide direct package URLs?
- Are pet files static data only?
- Is license information visible per pet?
- Are author and attribution fields available?
- Can package hashes be verified?
- Is mirroring allowed, or should entries remain link-only?
