# Pets

Mirrored pet packages belong here only after license review.

Suggested layout:

```text
pets/
  <source-id>/
    <pet-id>/
      pet.json
      spritesheet.webp
      meta.json
```

Do not add third-party assets here when the license is unclear. Use
`data/pets.json` with `status: "link-only"` instead.
